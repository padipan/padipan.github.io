ประวัติภาษาคอมพิวเตอร์

จุดเริ่มต้นของภาษาคอมพิวเตอร์

การเดินทางของภาษาคอมพิวเตอร์เริ่มต้นจากภาษาเครื่อง (Machine Language) สู่ภาษาระดับสูงที่เราใช้กันในปัจจุบัน

ยุคแรกเริ่ม (1940s - 1950s)

ในยุคแรกของคอมพิวเตอร์ การเขียนโปรแกรมทำได้โดยการใช้ภาษาเครื่อง (Machine Language) ซึ่งประกอบด้วยเลขฐานสอง (0 และ 1) เท่านั้น ต่อมาได้มีการพัฒนาภาษาแอสเซมบลี (Assembly Language) ที่ใช้คำสั่งที่อ่านง่ายขึ้น แต่ยังต้องเขียนคำสั่งในระดับต่ำอยู่

ภาษาระดับสูงภาษาแรก (1950s)

FORTRAN (Formula Translation) ถูกพัฒนาโดย IBM ในปี 1957 และถือเป็นภาษาโปรแกรมระดับสูงภาษาแรกที่ได้รับความนิยม โดยเฉพาะในงานด้านวิทยาศาสตร์และคณิตศาสตร์ ตามมาด้วย COBOL (Common Business-Oriented Language) ในปี 1959 ที่ถูกออกแบบมาสำหรับงานธุรกิจ

ยุคทองของภาษาโปรแกรม (1960s - 1970s)

ในช่วงนี้มีการพัฒนาภาษาโปรแกรมมากมาย เช่น BASIC ที่ถูกออกแบบมาให้ง่ายต่อการเรียนรู้สำหรับผู้เริ่มต้น, Pascal ที่เน้นการสอนหลักการเขียนโปรแกรมที่ดี และ C ที่พัฒนาโดย Dennis Ritchie ที่ Bell Labs ซึ่งกลายเป็นภาษาที่มีอิทธิพลมากที่สุดภาษาหนึ่ง

ยุคของการเขียนโปรแกรมเชิงวัตถุ (1980s - 1990s)

ภาษา C++ ถูกพัฒนาต่อยอดจาก C โดย Bjarne Stroustrup เพื่อรองรับการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming) ตามมาด้วย Java ที่พัฒนาโดย Sun Microsystems ซึ่งมีแนวคิด "Write Once, Run Anywhere" ทำให้โปรแกรมสามารถทำงานได้บนแพลตฟอร์มที่แตกต่างกัน

ยุคของเว็บและโมบาย (2000s - ปัจจุบัน)

ภาษาโปรแกรมในยุคนี้มุ่งเน้นไปที่การพัฒนาเว็บและแอปพลิเคชันบนอุปกรณ์เคลื่อนที่ เช่น JavaScript, Python, Ruby, PHP และล่าสุดคือภาษาที่เน้นประสิทธิภาพและความปลอดภัย เช่น Rust, Go และ Swift

เกร็ดความรู้

รู้หรือไม่? Ada Lovelace ถือเป็นโปรแกรมเมอร์คนแรกของโลก เธอได้เขียนอัลกอริทึมสำหรับเครื่องวิเคราะห์ (Analytical Engine) ของ Charles Babbage ในช่วงกลางศตวรรษที่ 19 ก่อนที่จะมีคอมพิวเตอร์เสียอีก

บุคคลสำคัญในประวัติศาสตร์คอมพิวเตอร์

Ada Lovelace (1815-1852)

Ada Lovelace ถือเป็นโปรแกรมเมอร์คนแรกของโลก เธอได้เขียนอัลกอริทึมสำหรับเครื่องวิเคราะห์ (Analytical Engine) ของ Charles Babbage ซึ่งเป็นเครื่องคำนวณเชิงกลที่ซับซ้อน เธอมองเห็นศักยภาพของเครื่องที่สามารถทำงานได้มากกว่าการคำนวณตัวเลข และได้เขียนโน้ตที่อธิบายวิธีการคำนวณลำดับ Bernoulli ซึ่งถือเป็นโปรแกรมคอมพิวเตอร์แรกของโลก

ผลงานสำคัญ: อัลกอริทึมแรกสำหรับเครื่องคำนวณ

Alan Turing (1912-1954)

Alan Turing เป็นนักคณิตศาสตร์ชาวอังกฤษที่ถือเป็นบิดาของวิทยาการคอมพิวเตอร์และปัญญาประดิษฐ์ เขาได้พัฒนาแนวคิดของเครื่องจักรที่สามารถจำลองการทำงานของอัลกอริทึมใดๆ ซึ่งรู้จักกันในชื่อ "Turing Machine" นอกจากนี้ เขายังมีบทบาทสำคัญในการถอดรหัสลับของนาซีในช่วงสงครามโลกครั้งที่ 2 ซึ่งช่วยให้ฝ่ายสัมพันธมิตรได้รับชัยชนะ

ผลงานสำคัญ: Turing Machine, การถอดรหัส Enigma

Grace Hopper (1906-1992)

Grace Hopper เป็นนักคณิตศาสตร์และนายพลในกองทัพเรือสหรัฐฯ เธอเป็นผู้บุกเบิกในการพัฒนาภาษาโปรแกรมระดับสูง และเป็นผู้พัฒนาคอมไพเลอร์ตัวแรกของโลก เธอมีส่วนร่วมในการพัฒนาภาษา COBOL ซึ่งเป็นภาษาโปรแกรมที่ใช้กันอย่างแพร่หลายในงานธุรกิจ นอกจากนี้ เธอยังเป็นผู้คิดค้นคำว่า "bug" ในความหมายของข้อผิดพลาดในโปรแกรมคอมพิวเตอร์

ผลงานสำคัญ: คอมไพเลอร์ตัวแรก, ภาษา COBOL

Dennis Ritchie (1941-2011)

Dennis Ritchie เป็นนักวิทยาการคอมพิวเตอร์ชาวอเมริกันที่มีชื่อเสียงจากการเป็นผู้สร้างภาษา C และมีส่วนร่วมในการพัฒนาระบบปฏิบัติการ UNIX ภาษา C ที่เขาพัฒนาขึ้นได้กลายเป็นภาษาโปรแกรมที่มีอิทธิพลมากที่สุดภาษาหนึ่ง และเป็นรากฐานของภาษาโปรแกรมอื่นๆ อีกมากมาย เช่น C++, Java, JavaScript, C# และอื่นๆ

ผลงานสำคัญ: ภาษา C, ระบบปฏิบัติการ UNIX
